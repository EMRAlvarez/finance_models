The cashflow_eir model provides a reliable way to calculate the cashflow, effective interest rates (EIR), net present values (NPV), and profit and loss (P&L) of a large number loans quickly. The model is a set of lightweight python scripts that can be run from any command prompt, terminal or application capable of executing python scripts.

Documentation

The Cashflow docs are split into three parts, each corresponding to their respective scripts.


model formulae data

The model script covers key computational functions for the Cashflow, EIR, NPV and P&L modelling. The majority of this script consists of the Cashflow class, which contains the the model itself in addition to other key methods required for outputting and visualising data.
